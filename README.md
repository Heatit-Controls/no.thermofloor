# Heatit App
This app adds support for the Z-wave and WiFi devices by [Heatit controls](http://www.heatit.com).  
For supported devices, see [Heatit Controls App for Homey](https://homey.app/no-no/app/no.thermofloor/Heatit-Controls/)  
Questions and feedback can be submitted to our [Forum post](https://community.homey.app/t/app-pro-heatit-controls-app-v3-8-8/49914) or via our contact form [Contact form](https://shop.heatit.com/contact)
